About This Project

This project uses a Convolutional Neural Net trained on a set of words to predict speech. The project was inspired from a tensorflow Kaggle competition.

See Competition Info Here



How the model works

The data used to train this model was roughly 1.5 GB of .wav files sampling 30 different words. There were roughly 1700 examples of each word. In order to run the .wav files through the model, they had to be first converted into a form that the model could analyze, in this case it was a .npy array. In order to do This the .wav files were first converted into MFCC (Mel-frequency cepstral coefficients) using the librosa python library. This conversion is visualized below.


Once the files were in MFCC form, they were numpy compatible and were saved into .npy arrays, the model was trained on these arrays.


Convnet Model Layout

The highest accuracy reached on the training set was 95%, however it was found to be overfit. After adding in a couple dropout layers and a max pooling layer, a score of 86% was achieved and was found to consistently label test data and user generated data correct. Below is a layout of the model.
